# Sistema Acadêmico
Projeto desenvolvimento em java utilizando os conceitos da Programação Orientada a Objetos.

# Preparando o Ambiente
Antes de tudo, é necessário ter a JRE instalada em sua máquina para que o computador consiga processar todo o sistema.

1. Clone o repositório ou baixe os arquivos;
2. Abra o projeto em uma IDE Java, como IntelliJ ou Eclipse.
3. Execute a classe `Program.java`.
4. Utilize o menu no console para interagir com o sistema.

# Abstração da Classe Pessoa
A Classe pessoa foi definida como abstrata pois ela representa apenas um conceito genérico geral do que é uma Pessoa. Não faria sentido criar uma Pessoa genérica, 
pois as pessoas podem ter diferentes tipos de nomes, idades, matrículas e entre outros.
